================================================================================
                        HOW TO USE THIS FEATURE
================================================================================

1. NAVIGATE TO REGISTRATION PAGE
   → Go to: http://localhost:3000/distributor/register

2. FILL THE FORM (3 Steps)
   Step 1 - Account Info: Name, Email, Phone, Password
   Step 2 - Company Info: Company Name, License, GST
   Step 3 - Address Info: Address, City, State, Pincode, Service Area

3. SUBMIT FORM
   → Click "Register" button
   → Form validates and submits to API
   → API creates user and distributor profile
   → Session is created and cookie is set
   → User is redirected to dashboard

4. VIEW DASHBOARD
   → See "Verification Pending" banner
   → View company information
   → Access quick action cards
   → Navigate to other pages
